# DSSAD 数据提取及分析软件 操作指导手册

## 一、软件概述

本软件用于读取符合 GB 44497-2024《智能网联汽车 自动驾驶数据记录系统》的 DSSAD（L3+ 数据记录）数据，提供车辆/设备参数展示、事件读取、视频播放、事件导出/Excel 转换/删除以及数据分析图表等功能。

## 二、系统要求

- 操作系统：Windows 7 SP1 / Windows 8 / Windows 10 / Windows 11（32 位或 64 位）
- 运行内存：≥ 4GB（推荐 8GB）
- 硬盘空间：≥ 2GB 可用空间
- 依赖环境：软件已打包为 exe，无需额外安装 Python 或 Qt

## 三、启动与登录

1. 双击运行 `DSSAD数据提取及分析软件.exe`。
2. 在登录窗口输入用户名和密码。
   - 默认管理员账号：`admin` / `admin`
3. 点击“登录”进入主界面。

## 四、主界面区域说明

### 1. 车辆/设备基本参数区域（顶部，非交互）
显示车辆 VIN、DSSAD 型号（I 型/II 型）、硬件型号、硬件序列号、软件版本号、IP 端口，以及各类事件（锁定/碰撞/碰撞风险/时间戳）的当前存储数量、已用容量、最大容量和总用量。

### 2. 加载/读取事件文件区域（左侧上方，交互）
- **连接 DSSAD**：输入 DSSAD 设备的 IP 地址和端口号，点击“连接 DSSAD”。
- **从 DSSAD 读取事件**：设置起止时间后点击“从 DSSAD 读取事件”，软件通过 DoIP/UDS 协议（本版本为模拟实现）读取事件列表。
- **读取本地事件文件**：选择单个 JSON 格式事件文件。
- **读取本地事件包**：选择一个包含多个事件子目录的导出包目录。

### 3. 事件列表区域（左侧中部，交互）
- 展示已读取的事件列表。
- 支持排序：全部事件时间倒序、时间戳事件时间倒序、碰撞事件时间倒序、碰撞风险事件时间倒序。
- 支持单选、多选、全选、取消全选。
- 单选事件时，右侧自动显示该事件的信息和视频。

### 4. 导出/转换/删除事件区域（左侧下方，交互）
- **导出**：将选定事件从 DSSAD/本地复制到上位机指定目录，生成车辆信息 JSON、事件元数据 JSON、数据点 JSON 及关联视频文件。
- **转换为 Excel**：将选定事件汇总为 Excel 列表，便于图表分析。
- **删除**：从 DSSAD 内部删除选定事件。**仅管理员（admin）可执行删除操作**，操作前会弹出二次确认。

### 5. 视频播放区域（右侧上方，交互）
- 支持多路摄像头切换（前视/后视/左视/右视/环视合成）。
- 提供播放、暂停、倍速（0.25x~4x）、逐帧前进/后退、可拖动时间轴、全屏功能。
- 播放未导出视频时，会提示先导出视频。
- 本地事件包会自动链接同目录下的视频文件。

### 6. 视频信息区域（右侧中部，非交互）
显示当前视频来源摄像头、记录起点时间、记录终点时间、画面当前时刻、总时长、总帧数、当前帧数、视频格式。

### 7. 事件信息区域（右侧中下部，非交互）
显示事件类型、是否锁定、事件类型编码、数据完整性、是否被篡改、经纬度、事件起点/终点时间等。
对于时间段事件，会随视频帧实时刷新相关数据（如车辆速度）。

### 8. 数据分析区域（右侧底部，交互）
- 选择纵轴数据字段，点击“添加图表”即可生成时间序列图。
- 图表窗口支持移动、缩放、关闭。
- 最多可同时打开 10 张以上图表。
- 时间戳事件不要求数据分析交互。

## 五、典型操作流程

### 流程 1：连接 DSSAD 并读取事件
1. 输入 IP 和端口，点击“连接 DSSAD”。
2. 输入起止时间，点击“从 DSSAD 读取事件”。
3. 在事件列表中选择目标事件。
4. 查看右侧事件信息和视频。
5. 点击“导出”或“转换为 Excel”保存到本地。

### 流程 2：分析本地已有事件包
1. 点击“读取本地事件包”，选择包含多个事件子目录的文件夹。
2. 在事件列表中选择一个事件。
3. 软件自动加载同目录下的视频文件并播放。
4. 在数据分析区域添加图表，查看速度、加速度等随时间变化曲线。

### 流程 3：生成图表报告
1. 单选一个时间段事件。
2. 在“数据分析区域”选择纵轴数据（如“车辆速度_km_h”）。
3. 点击“添加图表”，可重复添加多个字段。
4. 对图表截图或配合 Excel 输出使用。

## 六、GB 44497-2024 导出文件格式说明

软件导出目录结构如下：

```
DSSAD_EXPORT_YYYYMMDD_HHMMSS/
├── vehicle_info.json        # 车辆/设备基本信息
├── EVT-0001/
│   ├── metadata.json        # 事件元数据
│   ├── data_points.json     # 时序数据点
│   └── front.mp4            # 关联视频（如有）
└── EVT-0002/
    ├── metadata.json
    ├── data_points.json
    └── front.mp4
```

`metadata.json` 字段包括：事件 ID、事件名称、事件类型、事件类型编码、是否锁定、起止时间、经纬度、数据完整性、是否被篡改、视频/非视频文件列表、摄像头通道。

`data_points.json` 为时序数据数组，每个元素包含该时刻的车辆速度、横向/纵向加速度、踏板开度、方向盘转角、横摆角速度等字段。

## 七、示例数据

项目附带 `sample_data/` 目录，包含两个示例事件：
- EVT-0001：碰撞事件，含示例视频和 12 个数据点。
- EVT-0002：碰撞风险事件，含示例视频和 10 个数据点。

首次启动后，可通过“读取本地事件包”选择 `sample_data` 目录体验完整功能。

## 八、注意事项

1. 删除操作仅管理员可执行，且会二次确认，删除后不可恢复。
2. 视频文件需符合 mp4 格式；若系统缺少解码器，OpenCV 可能无法播放，可尝试用系统播放器打开导出的视频。
3. DSSAD 真实连接需要车辆支持 DoIP/UDS 协议，本版本提供了模拟数据用于功能演示和 UI 验证。
4. 所有时间显示均为世界时间 UTC+8，精确到毫秒。

## 九、常见问题

**Q1：登录失败怎么办？**  
A：请确认用户名和密码为 `admin` / `admin`。生产环境应替换为真实认证服务。

**Q2：连接 DSSAD 失败？**  
A：请确认车辆点火、网线/WiFi 连接正常、IP 和端口正确。本版本在没有真实设备时可通过读取本地示例数据体验功能。

**Q3：视频无法播放？**  
A：检查视频文件是否存在；可尝试导出后再用本地播放器查看。

**Q4：Excel 转换后中文乱码？**  
A：请使用 Microsoft Excel 或 WPS 打开，软件使用 UTF-8 编码生成 xlsx 文件。
